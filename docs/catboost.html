class catboost_data_engine.RealDataEngineCatboost(data_file, test_fraction=None)¶
  • Источник данных между real и моделями Catboost

    • При инициализации в случае неуспеха возвращает исключение от pandas.read_csv()

    • Делит выборку на обучающую и контрольную 1/test_fraction датчиком случайных величин

    • делает бинарную метку y_target

    • размеченный на Xtrain, ytrain, Xtest, ytest Dataframe создаётся в конструкторе,

    с ним можно работать из jupyter - Pool для catboost создается вызовом отдельного метода, catboost.Pool - это родной формат для catboost, данные в этом формате подаются на вход модели - Pool можно сохранить в файл, после этого убить обьект RealDataEngineCatboost, а на вход catboost подать имя файла с пулом

Parameters

data_file – path файла с выборкой,
