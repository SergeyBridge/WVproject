Коды¶

Здесь содержатся комментарии к программному коду с разбивкой по моделям:

  • источники данных для модели

  • гиперпараметры модели

  • параметры модели

Исходные данные из базы данных подаются на вход программы-адаптера, “источника данных”, который реализует связку исходные данные - модель

Источник данных:

  • преобразует исходные данные в БД pandas в соответствии с протоколом

  • проверяет входящие сырые данные на соответствие протоколу, возвращает протокольные ошибки, если воникают

  • из преобразованных данных формирует признаковое пространство для модели

  • преобразует признаковое пространство в родной формат модели для обучения

  • если требуется, преобразует признаковое пространство в формат модели для production (обычно этого не требуется, как правило, модели в продакшн используют pandas)

Гиперпараметры модели, как это подразумевается дальше в этом тексте, отдельно от параметров используются не всегда. Если не используется механизм GridSearch, GridSearchCV, как из sklearn, так и встроенные методы моделей для поиска по сетке (GridSearch), этот пункт будет пустым
